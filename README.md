# REVO Coffee Website

Сайт кофе, созданный по макету. Реализована семантическая верстка с использованием HTML5, стилизация с применением методологии БЭМ и адаптивная верстка с помощью Flexbox и Grid.

## Технологии

- HTML5
- CSS3 (БЭМ методология)
- Адаптивная верстка (Flexbox, Grid, медиазапросы)
- JavaScript (для мобильного меню и кнопки "Наверх")

## Особенности

- Фиксированная шапка при скролле
- Кнопка "Наверх" с плавной прокруткой
- Увеличенные touch-зоны для мобильных устройств
- Визуальный feedback при наведении и клике
- Шрифты соответствуют гайдлайнам HIG (мин. 17px)
- Полностью адаптивный интерфейс
