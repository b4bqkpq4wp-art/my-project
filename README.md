# REVO Coffee - Mobile Interface

Сайт кофе с полностью адаптивным мобильным интерфейсом.

## Технологии

- HTML5
- CSS3 (БЭМ методология, Flexbox, Grid)
- JavaScript
- Адаптивная верстка (Desktop First подход)

## Особенности реализации

- Полностью адаптивное меню с гамбургер-кнопкой
- Фиксированная шапка при скролле
- Кнопка "Наверх" с плавной прокруткой
- Размеры шрифтов соответствуют гайдлайнам HIG и MD
- Увеличенные touch-зоны для мобильных устройств
- Оптимизация для использования одним пальцем
